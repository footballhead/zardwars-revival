# ZardWars Revival

ZardWars encyclopedia currently hosted at http://zardwarsrevival.nfshost.com/

# Deploying

Requires PHP 5.6.

Copy the following to your host of choice:

* `encyclopedia/`
* `images/`
* `pages/`
* `screenshots/`
* `*.html`
* `*.php`
* `style.css`
* `sitemap.xml`
